# dapp-demo

Starcoin dapp demo.

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

I recommend vue-cli to start the build server.

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
